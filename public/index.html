Add something for others to draw

Make sure it follows these rules

  • Is 100 characters or less
  • Is not a pronoun
  • Is not personal to anyone
  • Is not reference that only you and your friends will understand
  • Is safe for work (this is a bit flexible)
  • Is something that others will actually be able to draw
  • Fits into the sentence, "you should draw..."

Please only flag things that deserve it

  • Things very NSFW
  • Offensive things
  • Something personal to someone
  • Things that aren't able to be drawn (e.g. 'I love my girlfriend')
  • Unknown pronouns
  • Nonsensical things
  • Links to things

Don't flag things because

  • You don't think it's good
  • You can't draw it
  • You keep seeing it (instead, add something new)
